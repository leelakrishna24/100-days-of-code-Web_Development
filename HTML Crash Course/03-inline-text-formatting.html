Things learnt - HTML Inline text formatting elements

  1. <strong> for Bold typeface should be ended with </strong> or <b> can also be used but not preffered much
  2. <em> for Italic typeface should be ended with <em> or <i> can also be used but not preffered much
  3. <sub> for Subscripting text (smaller size text and positioned below basline of regular text) should be ended with </sub>
  4. <sup> for Superscripting text (smaller text and positioned above baseline of regular text) should be ended with </sup>
  5. <mark> for highlighting text to visually standout from context should be ended with </mark>
  6. <small> for displaying text in smaller font size should be ended with </small>
  7. <ins> for underlining the text should be ended with </ins>
  8. <del> for mark the text that is removed or deleted from document should be ended with </del>
