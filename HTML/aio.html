Things learned

  1. HTML Elements

    1.     <tagname> "content goes here" </tagname>

      </tagname>
    2. <tagname> "content goes here" </tagname>

      <tagname>

  2. Formatting text HTML Elements

    1. <h4> for formatting text size should be ended with </h4>
    2. <h1> for formatting text size should be ended with </h1>
    3. <h2> for formatting text size should be ended with </h2>
    4. <h5> for formatting text size should be ended with </h5>
    5. <p> for starting an paragraph should be ended with </p>
    6. <h6> for formatting text size should be ended with </h6>
    7. <br /> for breaking the paragraph or to step into next line, it is self closing html element
    8. <h3> for formatting text size should be ended with </h3>
    9. <hr /> for horizontal line in html document, it is self closing element

  3. Inline Text formatting Elements

    1. <strong> for Bold typeface should be ended with </strong> or <b> can also be used but not preffered much
    2. <sup> for Superscripting text (smaller text and positioned above baseline of regular text) should be ended with </sup>
    3. <em> for Italic typeface should be ended with <em> or <i> can also be used but not preffered much
    4. <small> for displaying text in smaller font size should be ended with </small>
    5. <mark> for highlighting text to visually standout from context should be ended with </mark>
    6. <del> for mark the text that is removed or deleted from document should be ended with </del>
    7. <ins> for underlining the text should be ended with </ins>
    8. <sub> for Subscripting text (smaller size text and positioned below basline of regular text) should be ended with </sub>

  4. div and span tags

    1. <div> is used to define a division or a block-level container within a web page, should end with </div>
    2. <span> is used to define an inline-level container within a webpage, should end with </span>

  5. Lists

    1. Ordered lists starts with <ol> tag and ends with </ol> tag, in between consists list components in form of <li> context </li>
    2. Un-ordered lists starts with <ul> tag and ends with </ul> tag, in between consists list components in form of <li> context </li>
    3. Description lists starts with <dl> tag (description list), <dt> tag defines the term name, <dd> tag describes each term

  6. Images, Attributes

    1. <img> tag is used to embed an image in a web page.
      • It requires two attributes src of tag <src> to specify path of an image
      • Specifies an alternate text for image
    2. img tag doesn't have closing tag.

  7. Anchor tags, linking

    1. <a> tag is used to add links into HTML webpage they are also called as hyperlinks
    2. The context in between the anchor tags <a> and </a> upon clicking it, we are redirected to the stored link
    3. We can also link between two pages in our own folder
      • href is the attribute which indicates link's description.
      • The link text is the part that will be visible to the reader.
      • Clicking on the link text, will send the reader to the specified URL address
    4. The a elements has an attribute target which allows us to specify how to open document of url specified in href attribute by choosing defining value of target
      1. target = "_self" for opening in same tab
      2. target = "_blank" for opening in new window or tab

  8. Tables

    1. The <table> defines an HTML table, it should be closed with </table> as it is end tag.
      HTML table consists of <tr> <th> <td>
      1. <tr> element definas a table row, ends with </tr>.
      2. <th> element defines a table header, ends with </th>.
      3. <td> element defines a table cell, ends with </td>.

  9. HTML forms

    1. The HTML <form> element is used to create an HTML form for user input, it should be ended with </form>
    2. The HTML <input> element is the most used form element. it can be displayed in many ways, depending on the "type" attribute.
      • <input type="text" > Displays a single-line text input field
      • <input type="radio" > Displays a radio button (for selecting one of many choices)
      • <input type="checkbox" > Displays a checkbox (for selecting zero or more of many choices)
      • <input type="submit" > Displays a submit button (for submitting the form)
      • <input type="button" > Displays a clickable button
    3. The <label> tag defines a label for many form elements.
      • The <label> element is useful for screen-reader users,because the screen-reader will read out loud the label when the user focuses on the input element.
      • The <label> element also helps users who have difficulty clicking on very small regions (such as radio buttons or checkboxes) - because when the user clicks the text within the <label> element, it toggles the radio button/checkbox.
      • The for attribute of the <label> tag should be equal to the id attribute of the <input> element to bind them together.
    4. Radio Buttons
      • The <input type="radio"> defines a radio button.
    5. Checkboxes
      • The <input type="checkbox"> defines a checkbox
      • Checkboxes let a user select ZERO or MORE options of a limited number of choices.
    6. Submit Button
      • The <input type="submit"> defines a button for submitting the form data to a form-handler.
      • The form-handler is typically a file on the server with a script for processing input data.
      • The form-handler is specified in the form's action attribute.
    7. Name attribute for <input>
      • Notice that each input field must have a name attribute to be submitted.
      • If the name attribute is omitted, the value of the input field will not be sent at all.
