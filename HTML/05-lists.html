Things learnt - Lists

  • Item tag in lists starts with <li> ends with </li>

    • Types of lists

    1. Ordered lists starts with <ol> tag and ends with </ol> tag, in between consists list components in form of <li> context </li>
    2. Un-ordered lists starts with <ul> tag and ends with </ul> tag, in between consists list components in form of <li> context </li>
    3. Description lists starts with <dl> tag (description list), <dt> tag defines the term name, <dd> tag describes each term
